# Contributing

## How to Contribute

### 1. Report a Bug
- Open an Issue
- Include: tool name, what went wrong, screenshot

### 2. Suggest a New Tool
- Open an Issue with tag "enhancement"
- Describe the problem it solves
- How it would be used

### 3. Improve an Existing Tool
- Fork the repo
- Make your improvements
- Submit a PR with description

### 4. Add a New Tool
- Create a new HTML file in `/tools/`
- Include README for setup
- Submit PR for review

## Quality Standards

- ✅ No external dependencies (pure HTML/CSS/JS)
- ✅ Mobile responsive
- ✅ Accessible (keyboard nav)
- ✅ Under 500KB per tool
- ✅ No tracking/analytics

## Getting Started

1. Clone: `git clone https://github.com/daligao/ordinarymantrying-tools.git`
2. Make changes
3. Test locally
4. Submit PR
